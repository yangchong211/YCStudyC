# 本章导读 #

本章主要内容如下：

- `<mutex>` 头文件中的类及相关函数的介绍，包括互斥量、各种锁类型的介绍，以及 lock， try_lock，call_once 等函数的用法。
- `std::mutex` 与 Pthread 等线程库中的 `pthread_mutex_t` 类型的对比，包括两者编程接口的差异及其他细节。
- `std::mutex` 和 `std::lock_guard`、`std::unique_lock` 应用实例。
